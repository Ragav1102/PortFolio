## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Ragav1102/PortFolio.git
```

Navigate to the project folder:

```bash
cd PortFolio
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

or

```bash
npm run dev
```
